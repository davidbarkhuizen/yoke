write a python program that exhibits the follow behaviour

- prints integers from 1 to n
- for each number in the sequence, the program must apply specific replacement rules based on divisibility:
    * If the number is a multiple of 3, print "boom" instead of the number
    * If the number is a multiple of 5, print "bap" instead of the number
    * If the number is a multiple of both 3 and 5 (i.e., divisible by 15), print "boom-bap!"
    * For all other numbers, print the number itself
